# book-shopping-assistant

## 中文名 / Chinese Name
购书助手

## 英文名 / English Name
Book Shopping Assistant

## 一句话简介 / Tagline
**中文**：先查电子版，再比实体价，帮你省钱买书

**英文**：Check e-book first, compare prices next, save money on books

## 详细描述 / Description

**中文**：
给你一份书单，它能自动帮你找到最优购买方案。先查微信读书/得到等平台有没有电子版，能看就不用花钱买；需要实体书的，自动对比京东/当当/淘宝/拼多多四个平台的价格，还会分析当前距离 618 或双 11 还有几天，告诉你"现在买"还是"等大促更划算"。自动处理中文书别名（比如"不要让我思考"就是"点石成金"），不用再纠结书名搜不到。

**英文**：
Give it a book list, get the optimal purchase plan. First checks if books are available on WeRead/Dedao (save money by reading digitally). For physical books, compares prices across JD, Dangdang, Taobao, and Pinduoduo. Analyzes timing — tells you whether to buy now or wait for 618/Dual-11 sales. Auto-resolves Chinese book aliases (e.g., "不要让我思考" = "点石成金"), so you never miss a book due to naming differences.

## 功能亮点 / Key Features

### 中文

- **电子版优先**：自动查微信读书覆盖情况，能看电子版就不花冤枉钱买实体书
- **全网比价**：一键对比京东、当当、淘宝、拼多多四个平台的价格
- **促销时机分析**：智能分析当前距离 618/双 11 等大促的时间，给出购买时机建议
- **书名自动解析**：内置中文书名别名库，"不要让我思考"自动识别为"点石成金"
- **中英双语触发**：说"买书"或"buy books"都能唤起，中文用户和海外用户都好用
- **零依赖开箱即用**：纯 Python 脚本，无任何第三方库依赖

### English

- **E-book First**: Auto-checks WeRead coverage so you don't waste money on books you can read digitally
- **Cross-Platform Comparison**: Compare prices across JD, Dangdang, Taobao, and Pinduodao in one step
- **Smart Sale Timing**: Analyzes days until 618/Dual-11 and recommends the best time to buy
- **Auto Title Resolution**: Built-in Chinese book alias database — "不要让我思考" auto-resolves to "点石成金"
- **Bilingual Triggers**: Works with "买书" or "buy books" — great for both Chinese and international users
- **Zero Dependencies**: Pure Python, no third-party packages needed, works out of the box

## 适用场景 / Use Cases

- "帮我看看这几本书怎么买最划算"
- "这些书微信读书上有吗？没有的话淘宝上多少钱？"
- "我想买几本产品设计相关的书，帮我比价"
- "I want to buy these books, find the best deal"
- "618 快到了，现在是买书的好时机吗？"
